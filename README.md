## 2. Installation
    
* cloner le githup
```bash
git@github.com:C-eorl/P12-CRM.git
```
* créer environnement virtuel & activer
```bash
python3 -m venv .venv
source .venv/bin/activate
```
* démarrer l'application
```bash
python3 main.py
```
